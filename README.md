# My Lite Diary 

一款高颜值、轻量级且极度流畅的 Android 仿 iOS 风格私密日记软件。

---

## 项目特性

* **仿 iOS 极致美学**：精心雕琢的 UI 与交互设计，带给你媲美 iOS 原生的丝滑动效与视觉体验。
* **高度自定义**：界面样式随心定义，打造专属于你的个性化日记空间。
* **隐私万无一失**：内置强悍的日记加密功能，严密守护你的内心世界与私密数据。
* **导入与导出**：支持完整的日记数据导入与导出，无需担心数据绑定，迁移自由。
* **极致轻量**：体积占用极小，运行内存低，拒绝臃肿，回归写日记的纯粹。

## 技术栈与运行环境

* **开发语言**：Kotlin
* **适用平台**：Android 设备
* **IDE 推荐**：Android Studio (最新稳定版)
